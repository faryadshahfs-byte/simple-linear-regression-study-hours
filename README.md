# Simple Linear Regression

## Project Overview

This project demonstrates Simple Linear Regression using the study hours and Exam Marks dataset.

## Objective

Predict student exam marks based on study hours.

## Dataset

| Study Hours | Exam Marks |
| ----------- | ---------- |
| 1           | 45         |
| 2           | 50         |
| 3           | 55         |
| 4           | 60         |
| 5           | 65         |
| 6           | 70         |
| 7           | 75         |
| 8           | 80         |
| 9           | 85         |
| 10          | 90         |

## Technologies

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Google Colab

## Model

Linear Regression

### Equation

Exam Marks = 5 × Study Hours + 40

## Predictions

| Study Hours | Predicted Marks |
| ----------- | --------------- |
| 11          | 95              |
| 12          | 100             |
